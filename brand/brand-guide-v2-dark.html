Brand identity · v2.0 dark utility · est. 2004, rebuilt 2026

Skin over Steel

Twenty years ago this name lived on dark forums — training logs, rep power, private sections, earned trust. v2 keeps that DNA and rebuilds it as a modern utility application for men's sexual health: log-driven, evidence-graded, pseudonymous, and cheap to run. Not a content site. A tool with a community inside it.

02

Marks on dark

Same three marks as v1 — the system was built to invert. Copper brightens one step (#C97438) to hold contrast on steel; the hexagon stroke lightens so the monogram doesn't sink.

A · Wordmark. STEEL flips to bone-white; Skin stays copper. The warm voice is the only color on the page.

B · Monogram / app icon. The S counter goes gunmetal so the mark reads as punched metal. This is the favicon and PWA icon.

C · Signal mark. On dark it doubles as the app's loading state: dots pulse in Morse rhythm. Discreet to outsiders, unmistakable to members.

Cost-effective build path

The identity is engineered to run on a lean, mostly open-source stack — three system fonts' worth of Google Fonts, SVG marks, CSS tokens, no image-heavy design language. Indicative path from zero to utility app:

PhaseWhat shipsStackIndicative cost
1 · Open FloorEducation library, brand, waitlistStatic site (Astro/Eleventy) + free-tier hosting≈ $0–20/mo + domain
2 · The RoomPseudonymous community, signals, moderationSelf-hosted Discourse (open source), themed with these tokens≈ $20–50/mo VPS
3 · The LogProtocol tracker, exports, Signal ScoreLightweight app (SvelteKit/Next + Postgres via Supabase or similar)≈ $25–100/mo until scale

Design tokens in this document map 1:1 to CSS custom properties — the brand guide is literally the start of the stylesheet. Membership revenue (Vault) funds phases 2–3; no supplement affiliates required to keep the lights on.

07

Voice, unchanged — register, tuned

v1 voice rules stand: candid, evidence-led, unashamed, no hype, no shame, hard lines on dosing content. Dark utility adds one register shift — the app speaks like a good instrument.

Instrument voice (UI)

Terse, mono, factual. "Log saved. Week 9 of 12." · "3 sources attached." · "Export ready." The interface never cheers, congratulates with confetti, or guilt-trips a missed week. Respect is the tone.

Human voice (editorial & community)

Warm serif, first person welcome, direct about anatomy and feelings alike. The italic copper voice carries anything vulnerable. Members are members; asking for help is the smart move, said plainly.
